# Arch Hyprland Baseline

Personal baseline Arch + Hyprland setup.

Includes:
- Hyprland, Waybar, Rofi, Wlogout
- Ghostty terminal
- Gruvbox GTK + icons
- swww wallpaper setup
- SDDM theme override

Goal:
Restore a calm, usable system from stock Arch with minimal effort.

This repo is a *baseline*, not a full system snapshot.
